## Tasks Summary
-Task 1: Digital Literacy Awareness
Created an infographic on password security, internet usage, email etiquette, social media awareness, and cybercrime awareness using Canva.
-Task 2: Student Digital Portfolio
Created and refreshed profiles on GitHub, LinkedIn, and Kaggle to build professional online presence.
-Task 3: Coding & Collaboration Platforms
Solved a beginner's coding problem on HackerRank and created a Digital Literacy Quiz using Google Forms.
-Task 4: Professional Email & Etiquette
Created formal emails and developed a checklist on 'Do's and Don'ts' for social media usage.
-Task 5: Cybercrime Awareness
Created a case study on social media account hacking and developed steps to be followed for online safety.
